Run Locally

git clone https://github.com/jt5d/xrai-spatial-web.git
cd xrai-spatial-web && npm install
node scrape.mjs --serve
Open http://localhost:3210/spatial

Enable Jarvis Voice

ANTHROPIC_API_KEY=sk-xxx node scrape.mjs --serve
Click the mic button or type in the text field.
